# ☕ Java Inheritance Programs

This repository contains simple Java programs demonstrating the fundamental concepts of **Inheritance** in Object-Oriented Programming (OOP). These programs are designed for beginners to understand different types of inheritance using easy-to-follow examples.

---

## 📚 Programs Included

### 🔹 Single Inheritance
- Animal → Dog
- Vehicle → Car
- Employee → Developer

### 🔹 Multilevel Inheritance
- GrandFather → Father → Son

### 🔹 Hierarchical Inheritance
- Shape → Circle
- Shape → Square

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/jayachandiran-ux/Java-Inheritance.git
```

2. Open the project in Visual Studio Code or any Java IDE.

3. Compile the Java file.

```bash
javac FileName.java
```

4. Run the program.

```bash
java FileName
```
